# Requirements
 
 - [python:](https://www.python.org/downloads/) 3.10 or 3.11 or 3.12

# Setting up

```shell
python -m venv venv
```
## Activate the virtual environment
**For UNIX users**
```shell
source venv/bin/activate
```
**For Win users**
```shell
.\venv\Scripts\activate
```
## Install dependencies

```shell
pip install -r requirements.txt
```
